### 苹果更新开发者程序许可协议导致问题

* 前两天在打包的时候突然除了这个错，后来发现原来是苹果更新了开发者程序协议

```
The 'Apple Developer Program License Agreement' has been updated. In order to access certain membership resources, you must accept the latest license agreement. ...

```
